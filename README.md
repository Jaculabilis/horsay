# horsay
Takes a line of input and formats it in a text bubble next to a horse.

```
$ horsay
Hello, world!
     ./|,,/|   
    <   o o)   ╭───────────────╮
   <\ (    | ──┤ Hello, world! │
  <\\  |\  |   ╰───────────────╯
 <\\\  |(__)   
<\\\\  |       
$ echo Hello, world! | horsay
     ./|,,/|   
    <   o o)   ╭───────────────╮
   <\ (    | ──┤ Hello, world! │
  <\\  |\  |   ╰───────────────╯
 <\\\  |(__)   
<\\\\  |       

```
